How did you apply iterative design in your project?

Aplicamos diseño iterativo porque aunque la pagina ya estaba creada, nosotros seguimos agregándole cosas y dandole mantenimiento. Como el diseño iterativo es cíclico, es el enfoque que le dimos a nuestra pagina web para poder darle mantenimiento en el futuro y poder hacer actualizaciones.

Create your own methodology to develop a user interface. It’s important to justify all the steps.

Nosotros usaremos una metodología iterativa para nuestra user interface, de esta manera sera mas fácil adaptarla a lo que el usuario quiere ver.

  1. Crea el template para nuestra user interface.
  2. Tomar un grupo selecto de personas para que interactúan con la user interface y estudiar sus reacciones así como preguntarles su opinion sobre la user interface.
  3. En base a la información recopilada de los usuarios que estudiamos, modificar la user interface para hacerla mas atractiva y que cumpla con todas las expectativas de los usuarios.
  4. Seguir aplicando estos 3 pasos hasta obtener los resultados deseados.

In your project did you follow the first three laws of simplicity? How?

Reduce: Hicimos nuestra pagina de modo que sea lo mas fácil de usar al reducir al máximo la funcionalidad. Nuestra pagina consiste en su mayoría en puro texto y es muy intuitiva, tiene muy pocos botones lo que la convierte en una herramienta muy intuitiva para el usuario.

Organize: Nuestra pagina cuenta solo con la información que es realmente esencial para no confundir al usuario ni agobiarlo con tanto texto. Cada sección de nuestra pagina esta separada por bloques de diferentes colores, esto nos resulta muy conveniente para organizar la información y para que el usuario pueda entenderla mas fácil.

Time: Es muy frustrante para todas las personas tener que esperar o entrar a paginas web para buscar información pero que en vez de proporcionar la información solo le dan vueltas al asunto y te deja igual que como empezaste. Nuestra pagina elimina este problema ya que al estar separada por colores y al tener solamente la información realmente necesaria, le ahorramos mucho tiempo al usuario.
